# 🎥 视频录制指南

## 快速录制（推荐）

### 使用 QuickTime（macOS）

1. **打开 QuickTime Player**
2. **文件 → 新建屏幕录制**
3. **点击录制按钮**
4. **选择录制区域**（选择浏览器窗口）
5. **开始演示**

### 演示流程（2-3 分钟）

**0:00-0:15 - 介绍**
- 打开 https://binance-psychology.vercel.app/
- 展示主页标题和副标题
- 说明：AI 驱动的交易心理分析工具

**0:15-0:30 - 问题说明**
- 滚动页面
- 说明：80% 交易亏损来自心理问题
- FOMO、报复性交易、止损不及时

**0:30-1:00 - 填写表单**
- 输入 API Key（打码显示）
- 选择 ETHUSDT
- 选择 365 天
- 点击"开始分析"

**1:00-1:30 - 分析结果**
- 展示核心统计（总交易、胜率、盈亏）
- 展示心理健康评分
- 展示对比分析

**1:30-2:00 - 数据可视化**
- 展示时段胜率图表
- 展示持仓时间分布
- 说明：找出最佳交易时段

**2:00-2:30 - 问题诊断**
- 展示发现的问题（FOMO、报复性交易）
- 展示改进建议

**2:30-3:00 - 实时监控**
- 点击"启动实时监控"
- 展示监控面板
- 说明：实时价格监控和 FOMO 警告

**3:00-3:10 - 结束**
- 展示 GitHub 链接
- 感谢 Binance

### 录制完成后

1. **导出视频**
   - 格式：MP4
   - 分辨率：1920x1080
   - 帧率：30fps

2. **上传到 YouTube**
   - 标题：Binance Trading Psychology Coach - AI 交易心理分析工具
   - 描述：Build with Binance 2025 参赛项目
   - 标签：Binance, Trading, Psychology, AI, OpenClaw

3. **获取链接**
   - 复制 YouTube 链接
   - 更新 README.md

## 或者使用 Loom

1. 访问 https://www.loom.com/
2. 安装浏览器插件
3. 点击录制
4. 选择"屏幕 + 摄像头"或"仅屏幕"
5. 开始演示
6. 录制完成后自动上传
7. 获取分享链接

## 简化版（如果时间紧）

**只录制核心功能（1 分钟）：**
1. 打开网站（5秒）
2. 填写表单（10秒）
3. 展示分析结果（30秒）
4. 展示实时监控（15秒）
