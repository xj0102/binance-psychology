# 🎬 Demo 演示

## 项目介绍

**Binance Trading Psychology Coach** - AI 驱动的交易心理分析工具

### 核心价值

研究表明，**80% 的交易亏损来自心理问题**，而非技术分析错误：
- 📈 FOMO（追涨杀跌）
- ⚡ 报复性交易（亏损后急于翻本）
- 🛑 止损不及时（让亏损扩大）
- ⏰ 疲劳交易（深夜决策质量差）

本工具通过数据分析，帮你识别并改正这些问题。

## 功能演示

### 1. OpenClaw 集成 🦞

**原生支持对话式交互：**

```
用户: "分析我的 BTC 交易心理"
AI: 正在分析你的交易数据...
    
    📊 分析结果：
    - 总交易: 17 笔
    - 胜率: 11.8%
    - 发现 4 次报复性交易
    - 平均止损点 26.2%（建议 3%）
    
    💡 建议：
    1. 亏损后等待 1 小时再交易
    2. 设置 -3% 自动止损
    3. 避免深夜交易
```

### 2. Web 界面

**简洁直观的分析界面：**

1. **输入 API Key**（只需只读权限）
2. **选择交易对**（BTC/ETH/BNB/SOL）
3. **选择时间范围**（7/30/90/180/365 天）
4. **点击分析**

### 3. 分析结果

**核心统计：**
- 总交易次数
- 胜率
- 总盈亏
- 盈亏比

**心理健康评分：**
- 0-100 分评分系统
- 动态进度条
- 三级评级（优秀/良好/需改进）

**对比分析：**
- 你 vs 市场平均胜率（45%）
- 你 vs 市场平均盈亏比（1.5）
- 你 vs 市场平均 FOMO 频率（15%）

### 4. 数据可视化

**时段胜率图表：**
- 柱状图展示每小时胜率
- 找出最佳/最差交易时段

**持仓时间分布：**
- 饼图展示短线/中线/长线占比
- 优化交易节奏

### 5. 问题诊断

**自动识别：**
- 🔴 FOMO 追涨（价格暴涨后买入）
- ⚡ 报复性交易（亏损后 30 分钟内交易）
- 🛑 止损纪律较差（大额亏损 >5%）

### 6. 实时预警 🚨

**启动监控后：**
- WebSocket 实时价格监控
- 价格暴涨 >5% 立即警告
- 显示历史 FOMO 胜率
- 建议等待回调

**警告示例：**
```
🚨 FOMO 警告！

BTCUSDT 刚涨了 6.2%

你历史上有 5 次 FOMO，胜率 0%

建议：等待回调再买入
```

### 7. 历史记录

- 自动保存分析记录
- 最近 10 条历史
- 评分趋势追踪

### 8. PDF 导出

- 一键导出完整报告
- 打印友好格式
- 包含所有核心数据

## 真实案例

**用户 A 的 ETH 交易分析：**

**问题：**
- 胜率：5.9%（17 笔只赢 1 笔）
- 4 次报复性交易
- 止损点 26.2%（应该 3%）
- 全在深夜交易（23:00）

**使用工具后：**
- 设置 -3% 自动止损
- 亏损后等待 1 小时
- 避免深夜交易
- **预计胜率提升至 45%+**
