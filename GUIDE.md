# 币安交易心理分析助手 - 使用指南

## 快速开始

### 1. 安装

```bash
bash install.sh
```

### 2. 配置 API

登录币安 → 账户 → API 管理 → 创建 API

**重要：只勾选"读取"权限！**

```bash
export BINANCE_API_KEY="your_api_key"
export BINANCE_API_SECRET="your_api_secret"
```

### 3. 运行分析

**快速分析（最近 30 天）：**
```bash
node analyze.js BTCUSDT 30
```

**详细分析（最近 1 年）：**
```bash
node detailed-report.js ETHUSDT 365
```

**演示模式（无需 API）：**
```bash
node demo.js
```

## 技术细节

### 心理模式识别算法

**1. FOMO 检测**
- 检测买入前 1 小时价格涨幅
- 涨幅 > 5% 且最终亏损 = FOMO

**2. 报复性交易**
- 检测亏损后 30 分钟内的交易
- 仓位增加 > 30% = 报复性交易

**3. 时段分析**
- 统计每个小时的胜率
- 找出最佳/最差交易时段

**4. 止损纪律**
- 计算平均止损点
- 统计大额亏损（> 5%）次数
