# 币安交易心理分析助手 🧠

## 这是什么？

一个帮你发现交易坏习惯的 AI 助手。

不是告诉你买什么币，而是告诉你**为什么总是亏钱**。

## 它能发现什么？

- 🔴 **FOMO（追涨）** - 你是否总在暴涨后追涨？
- 🔴 **报复性交易** - 亏损后是否立即加仓"翻本"？
- 🔴 **时段问题** - 哪个时段你的决策最差？
- 🔴 **止损纪律** - 你是否果断止损？
- 🔴 **持仓时间** - 短线还是长线更适合你？

## 快速开始

### 1. 配置 API

```bash
export BINANCE_API_KEY="your_api_key"
export BINANCE_API_SECRET="your_api_secret"
```

**注意：只需要"读取"权限，不需要交易权限！**

### 2. 分析交易

```bash
node analyze.js BTCUSDT 7
```

参数：
- `BTCUSDT` - 交易对
- `7` - 分析最近 7 天

### 3. 查看报告

```
📊 BTCUSDT 交易心理分析

时间范围: 最近 7 天
总交易: 25 笔
胜率: 36%
盈亏: $-500

🔴 发现的问题:

1. FOMO（追涨）
   15 次在暴涨后追涨，胜率 20%
   平均亏损 $50/笔

2. 报复性交易
   8 次亏损后立即加仓，胜率 12%

3. 时段问题
   22:00 胜率最低（20%）
   09:00 胜率最高（60%）

4. 止损纪律: 较差
   5 次亏损超过 5%
   平均止损点: 8.5%

💡 改进建议:

- 避免追涨: 价格 1 小时涨幅 > 5% 时等待回调
- 亏损后休息: 强制休息 1 小时再交易
- 调整时段: 避开 22:00，多在 09:00 交易
- 严格止损: 设置 -3% 自动止损
```
