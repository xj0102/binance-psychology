## 项目简介

这是一个基于 AI 的交易心理分析工具，通过分析用户的 Binance 交易历史，识别情绪化交易模式（如 FOMO、报复性交易、止损纪律问题），并提供可视化报告和改进建议。

### 核心功能

1. **心理模式识别**
   - FOMO 交易检测（价格暴涨后追高）
   - 报复性交易识别（亏损后立即再次交易）
   - 止损纪律分析（平均止损幅度）

2. **数据可视化**
   - 交易时间分布图表
   - 持仓时长分析
   - 盈亏统计

3. **心理健康评分**
   - 0-100 分评分系统
   - 三档评级（优秀/良好/需改进）
   - 与市场平均水平对比

4. **实时监控**
   - WebSocket 实时监控新交易
   - FOMO 预警提醒
   - 异常交易模式告警

5. **OpenClaw 集成**
   - 对话式交互（主要使用方式）
   - 自然语言查询交易心理状态
   - AI 助手主动提醒

### 技术栈

- **前端：** HTML5, CSS3, JavaScript, Chart.js
- **后端：** Node.js, Express
- **API：** Binance REST API, WebSocket API
- **存储：** LocalStorage
- **AI 框架：** OpenClaw Skill Framework
- **部署：** Vercel

### 使用场景

1. **交易复盘**：分析历史交易，发现心理问题
2. **实时监控**：交易时实时提醒情绪化决策
3. **长期改进**：追踪心理健康评分变化趋势

### 创新点

- **心理学视角**：从心理学角度分析交易行为，而非单纯技术分析
- **对话式交互**：通过 OpenClaw 实现自然语言交互，降低使用门槛
- **实时预警**：主动监控并提醒用户避免情绪化交易
- **隐私保护**：只需 API 读取权限，不涉及交易操作

### 目标用户

- 频繁交易的散户投资者
- 容易情绪化决策的交易者
- 希望提升交易纪律的用户

## 安全性

- 仅需 Binance API 的**读取权限**
- 不涉及任何交易操作
- API 密钥本地存储，不上传服务器
