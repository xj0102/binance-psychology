## 报名表信息

**作品名称：**
币安交易心理分析助手

**功能亮点：**
- 自动分析交易历史，发现 FOMO、报复性交易等心理模式
- 统计不同时段的胜率，找出最佳/最差交易时间
- 评估止损纪律，给出改进建议
- 帮助交易员认识自己，避免情绪化决策

**技术实现：**
- 币安 API 获取交易历史
- 算法识别心理模式
- 生成个性化报告

**差异化：**
不是交易工具，而是心理教练。帮助用户成长，而不是替用户交易。
