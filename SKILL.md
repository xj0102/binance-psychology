# 币安交易心理分析助手

## 这不是交易工具，而是你的心理教练

通过分析你的交易历史，发现隐藏的心理模式，帮助你避免情绪化决策。

## 核心功能

### 1. FOMO 检测 🔴
发现你在价格暴涨后追涨的次数和胜率。

### 2. 报复性交易识别 🔴
检测你在亏损后是否立即加大仓位"翻本"。

### 3. 时段分析 📊
统计你在不同时段的胜率，找出最佳/最差交易时间。

### 4. 持仓时间分析 ⏱️
分析你的短线 vs 长线胜率，找到最适合你的交易周期。

### 5. 止损纪律评估 🛑
检查你是否果断止损，还是总让亏损扩大。

```

```
